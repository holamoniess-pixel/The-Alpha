# The Alpha

AI-powered automation system with voice control, vision, learning, and multi-agent orchestration.

## Quick Install (One-Liner)

### Linux
```bash
curl -sSL https://raw.githubusercontent.com/holamoniess-pixel/The-Alpha/main/install.sh | bash
```

### Termux (Android)
```bash
curl -sSL https://raw.githubusercontent.com/holamoniess-pixel/The-Alpha/main/install-termux.sh | bash
```

### Windows (PowerShell)
```powershell
irm https://raw.githubusercontent.com/holamoniess-pixel/The-Alpha/main/install.ps1 | iex
```

### macOS
```bash
curl -sSL https://raw.githubusercontent.com/holamoniess-pixel/The-Alpha/main/install.sh | bash
```

## Features
- Voice recognition & synthesis
- Computer vision
- Multi-agent orchestration
- Automation engine
- Learning system
- Security vault

- Python 3.8+
